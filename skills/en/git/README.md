# Git

Skills for **Git integration, quality gates, and review response** across branch sync, pre-PR checks, and post-PR handling.

## Skills

- `recipe-merge-repair`: compare an upstream branch against the current recipe branch, adapt the recipe, and land a safe merge after major upstream updates.
- `pr-gate`: checks before opening/updating a PR (diff review, docstrings, typing cleanup, lint/test gates).
- `pr-feedback`: handles reviewer comments after a PR is open (triage, fixes, replies, re-validation).
- `pr-skill-review`: review a PR that modifies skill files (apply skill review checklist, output issues/suggestions and what to keep).
