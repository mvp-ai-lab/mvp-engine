# Git

与 **Git 集成、质量门禁、评审响应** 相关的 skills，覆盖分支同步、PR 前检查和 PR 后处理。

## Skills

- `recipe-merge-repair`: 上游分支有重大更新时，先比较分支差异，再适配 recipe，并把上游分支安全 merge 到当前开发分支。
- `pr-gate`: 提交/更新 PR 前检查（差异审阅、docstring 补齐、typing 修正、lint/test 门禁）。
- `pr-feedback`: PR 创建后处理 reviewer 评论（分拣、修复、回复、回归验证）。
- `pr-skill-review`: 对「修改 skill 文件」的 PR 做评审（应用 skill 评审清单，输出问题/建议与可保留项）。
